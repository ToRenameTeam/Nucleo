# Business Requirements

| BR1 | Reduction of Administrative Fragmentation | The project must reduce healthcare administrative fragmentation by centralizing data management and booking processes. |
| --- | ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| BR2 | Operational Efficiency Improvement        | The system must improve efficiency in booking and health documentation management compared to current processes.       |
| BR3 | Increased User Satisfaction               | The system must increase the satisfaction level of both patients and doctors through process simplification.           |
| BR4 | Family Data Centralization                | The system must centralize the health data of all members of a family unit into a single platform.                     |
| BR5 | Administrative Workload Reduction         | The system must reduce administrative workload for both patients and healthcare operators.                             |
