# Non-Functional Requirements

| ID  | Requirement                  | Description                                                                                       |
| --- | ---------------------------- | ------------------------------------------------------------------------------------------------- |
| NF1 | Concurrent Access Management | The system must support multiple concurrent accesses without significant performance degradation. |
| NF2 | Standard Format Support      | The system must support standard formats for health documents (PDF).                              |
| NF3 | Testability                  | The automatic test coverage (unit and integration) of the code must be at least 75%.              |
